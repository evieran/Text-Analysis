# Text-Analysis

## Description

This project contains Python scripts to perform text analysis on novels or large text files. The analysis includes tasks such as tokenization, word clouds, sentiment analysis, and topic modeling. The code provided in this repository is for educational purposes and helps understand various Natural Language Processing (NLP) techniques.

## Features

- Text Cleaning and Preprocessing
- Creating Word Clouds
- Sentiment Analysis using VADER
- Topic Modeling using LDA

## Dependencies

- nltk
- gensim
- wordcloud
- pandas
- matplotlib
- PIL
- numpy

## Code Structure

- **Part 1:** Reading and preprocessing the text file.
- **Part 2:** Data cleaning and visualization.
- **Part 3:** Advanced word cloud and sentiment analysis.
- **Part 4:** Text cleaning, tokenization, and topic modeling.

## Note

The entire code provided in this repository is intended solely for educational purposes.
